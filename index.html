My Projects

Zaza Barbershop

A responsive and elegantly designed one-page website created to establish a professional online presence beyond social media platforms. Designed with user experience in mind, it features a clean layout and intuitive navigation to ensure accessibility across all devices. The site serves as a central hub, showcasing key information and engaging visuals, while reinforcing brand identity. By stepping outside the limitations of social media, this website provides greater flexibility and control over content, allowing for a more personalized and lasting impression on visitors.

Technologies: HTML, CSS, Bootstrap, JavaScript, Form Submit

Globcipe

Globcipe is a web app created by a team of four in under two weeks as our final project for Le Wagon Bootcamp. The project required strong teamwork, clear communication, and effective use of tools like Trello and GitHub for collaboration. Each member contributed unique skills, and we worked together to solve challenges and manage tight deadlines, gaining valuable technical and teamwork experience.

Technologies: HTML, CSS, Bootstrap, JavaScript, Ruby, Ruby on Rails, Heroku, Git, GitHub, Figma

Gnome Garden

Gnome Garden is a fun take on an Airbnb clone, created as a practice web app before our project weeks at Le Wagon. This exercise helped us build a strong foundation in the technical and collaborative skills needed for our final project. Working as a team, we focused on implementing key features like user authentication and booking systems while gaining hands-on experience with version control and Agile workflows. This project was an essential stepping stone in refining our coding and teamwork abilities.

HTML, CSS, Bootstrap, JavaScript, Ruby, Ruby on Rails, Figma, Heroku, Git, GitHub, Figma
