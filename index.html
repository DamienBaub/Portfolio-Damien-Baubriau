My Projects

Zaza Barbershop

A responsive and elegantly designed one-page website created to establish a professional online presence beyond social media platforms. Designed with user experience in mind, it features a clean layout and intuitive navigation to ensure accessibility across all devices. The site serves as a central hub, showcasing key information and engaging visuals, while reinforcing brand identity. By stepping outside the limitations of social media, this website provides greater flexibility and control over content, allowing for a more personalized and lasting impression on visitors.

Technologies: HTML, CSS, Bootstrap, JavaScript, Form Submit, Figma

Globcipe

Globcipe is a web app created by a team of four in under two weeks as our final project for Le Wagon Bootcamp. The project required strong teamwork, clear communication, and effective use of tools like Trello and GitHub for collaboration. Each member contributed unique skills, and we worked together to solve challenges and manage tight deadlines, gaining valuable technical and teamwork experience.

Technologies: HTML, CSS, Bootstrap, JavaScript, Ruby, Ruby on Rails, Heroku, Git, GitHub, Figma

Todo List

This To-Do List project is a simple web application for managing tasks effectively. Built with HTML, CSS, and JavaScript, it features an intuitive interface for adding, editing, and deleting tasks, along with marking them as completed. The app uses local storage to persist tasks and is fully responsive for seamless use on any device. This project highlights key front-end skills, including DOM manipulation and event handling, in a clean, minimalistic design.

Technologies: HTML, CSS, JavaScript

Stimulus JS library

The Stimulus JS library project demonstrates the integration of Stimulus JS to create dynamic and interactive web pages. This project showcases how Stimulus enhances traditional HTML by connecting JavaScript behaviors directly to the DOM, enabling features like real-time updates and user interaction without overcomplicating the code. It highlights the simplicity and efficiency of Stimulus for managing state and events, offering an elegant solution for building modern, responsive, and maintainable web applications.

Technologies: HTML, CSS, JavaScript, Stimulus JS
